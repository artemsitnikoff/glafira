# Промт генерации уточняющих вопросов (П.2 — режим «по слабым сторонам резюме»)

Вопросы для авто-QA (фича «Уточняющие вопросы и автоперевод», режим `auto_qa_mode='weak'`)
НЕ генерируются отдельным запросом — они создаются КАК ЧАСТЬ скоринга кандидата
(`score_candidate` → поле `questions` в ответе LLM, сохраняется в `AiEvaluation.questions`).
То есть это секция общего скоринг-промта (`prompts.py::SCORING_SYSTEM_PROMPT_BASE`,
дублируется в `scoring_system.md`).

Когда в карточке включена авто-QA в режиме «по слабым сторонам резюме», Глафира берёт
эти сгенерированные при скоринге вопросы и отправляет их кандидату в чат (один раз).

## Точная инструкция в промте (часть схемы ответа скоринга)

Поле схемы:
```
"questions": ["вопрос 1", "вопрос 2", "вопрос 3", "вопрос 4", "вопрос 5"]
```

Блок «ВАЖНО по вопросам»:
```
ВАЖНО по вопросам:
- Создай РОВНО 5 самых релевантных вопросов для первого контакта с кандидатом
- Вопросы должны быть конкретными репликами рекрутёра для разговора с кандидатом
- Приоритизируй микс: пробелы резюме + выявленные минусы/риски
- Каждый вопрос — это то, что рекрутёр реально спросил бы при собеседовании
- Формат: плоский массив из 5 строк (НЕ dict), максимум 5 вопросов
```

Ключевое: вопросы приоритизируют **пробелы резюме + выявленные минусы/риски** (`risks`),
то есть «слабые стороны». Контекст для модели — описание вакансии + данные кандидата
(тот же ввод, что и для оценки) + особые указания рекрутёра (`recruiter_instructions`).

## Где используется
- Генерация: `services/glafira/prompts.py` (часть `SCORING_SYSTEM_PROMPT_BASE`), вызывается из `scoring.score_candidate`.
- Отправка: `services/glafira/auto_qa.py::ask_auto_qa_questions` — берёт `AiEvaluation.questions`
  последней оценки заявки, оборачивает в сообщение `_compose_questions_message` и шлёт в hh-чат один раз.

## Альтернатива (режим «определённые вопросы», `auto_qa_mode='fixed'`)
LLM НЕ участвует: отправляется статический текст `vacancy.auto_qa_fixed_text` (рекрутёр
пишет сам или вставляет из шаблона сообщений) — всегда одни и те же вопросы.
